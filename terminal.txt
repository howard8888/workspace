[llm-help] OpenAI / LLM API setup
  1. Install the Python SDK into this SAME Python environment:
       python -m pip install --upgrade openai
  2. Verify which version this interpreter sees:
       python -c "import openai; print(openai.__version__)"
  3. Save your API key in menu 48, or otherwise set OPENAI_API_KEY in the environment.
  4. Optionally set a default smoke-test model in menu 48.
     - CCA8 stores this in CCA8_OPENAI_MODEL.
     - If unset, CCA8 defaults to gpt-5.4.
  5. Run the smoke test from menu 48 to verify:
       - the SDK imports correctly,
       - OPENAI_API_KEY is present,
       - the configured model name is being used, and
       - an actual API call succeeds.
  Notes:
       - Use 'python -m pip ...' so pip targets the same interpreter running CCA8.
       - On Windows, setx affects future cmd.exe windows, not the current one.
       - This menu also sets os.environ in the current CCA8 run, so testing can work immediately.
       - You need both a valid API key and API billing/credits for live calls.
